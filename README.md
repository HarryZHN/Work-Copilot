# Work Copilot

一个简洁高效的工作辅助工具，目前集成备忘录和排期管理功能。


## 🛠️ 技术栈

- **框架**: Vue 3 + TypeScript
- **构建工具**: Vite 5
- **路由**: Vue Router 4
- **数据库**: Dexie.js (IndexedDB)


## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 📖 使用说明

### 备忘录功能

1. 在左侧导航点击「备忘录」进入备忘录页面
2. 点击「新建备忘录」按钮创建新备忘录
3. 填写标题和内容后点击「保存」
4. 使用搜索框快速查找备忘录
5. 点击备忘录卡片可编辑或删除

### 排期管理

1. 在左侧导航点击「排期表」进入排期页面
2. 点击日历上的日期创建任务
3. 设置任务标题、详细内容和日期范围
4. 使用「今天」按钮快速回到当前日期
5. 点击任务可编辑或删除，勾选复选框标记完成状态
