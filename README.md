# Work Copilot

一个简洁高效的工作辅助工具，目前集成备忘录和排期管理功能，纯本地存储。\
🔗 可用地址：<https://workcopilot.netlify.app>

## 🛠️ 技术栈

- **框架**: Vue 3 + TypeScript
- **构建工具**: Vite 5
- **路由**: Vue Router 4
- **数据库**: Dexie.js (IndexedDB)

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```
